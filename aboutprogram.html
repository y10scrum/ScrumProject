Tommy's Timer

About the Game

Tommy's Timer is a free to play program with no sign up required, all you need to do is visit the "Game Test" tab in the navigation bar at the top of the screen, instructions on how to play the game will be posted. Tommy's Timer has many innovative features, with more to come! As of yet, the program boasts multiple user skins, a plethora of background images, a score system based on time, along with a way to save high scores. Also, there is a user friendly start game interface, along with a intuitive pause menu for when you need to go eat but you're in the middle of a game. Features to come in the future will be a two player option, randomly moving badguys, and a variety of powerups for the main character. On top of this, the game's graphics will be improved, with a variety of more unlockable character, along with a downloadable version of the game so the user does not always need to visit the website to play the game everytime. The Simon Game at the bottom of the game test tab is another smaller project that is a simple rinse & repeat game that requires you to match the pattern of the AI, or "Simon". This game however is still under development and will be ready shortly.
